Equity Portfolio Analyzer

Hello There!

This project was made to perfect my knowledge of VBA and Excel as they are important tools for data analysis. The program helps to make investment decisions as it shows the performance of a stock portfolio in the past and compares it with different scenarios.

The program analyzes portfolio build with Wig20 companies (polish stock market) and enables the user to see its performance in the past.
